# Activitat de vores:

Utilitzant com a base aquest document HTML has de fer el CSS corresponent.

```
<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Targetes amb Border</title>
  <link rel="stylesheet" href="estils.css">
</head>
<body>
  <h1>Targetes amb estil de vora</h1>

  <div class="targeta targeta1">
    <h2>Targeta 1</h2>
    <p>Vora sòlida i cantonades arrodonides.</p>
  </div>

  <div class="targeta targeta2">
    <h2>Targeta 2</h2>
    <p>Vora discontínua amb colors diferents per cada costat.</p>
  </div>

  <div class="targeta targeta3">
    <h2>Targeta 3</h2>
    <p>Vora doble i forma gairebé circular.</p>
  </div>

</body>
</html>
```

El CSS ha de tenir:

- Una classe targeta que tingui un color de fons que tu vulguis, una amplada de 500 pixels i una alçada de 400 pixels.
- Una classe targeta1 amb vora sòlida i cantonades arrodonides.
- Una classe targeta2 amb vora discontínua amb colors diferents per cada costat.
- Una classe targeta3 amb vora doble i forma gairebé circular.
