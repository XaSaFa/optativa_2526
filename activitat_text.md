# Bateria d'exercicis de text - per corregir a classe


## 1. Alineació del text
- 1,1 text-align
  - Crea un <h1> alineat al centre.
  - Alinea un paràgraf <p> a la dreta.
  - Crea tres paràgrafs i aplica:
    - El primer alineat a l’esquerra.
    - El segon centrat.
    - El tercer justificat.
- 1.2 text-align-last
  - Fes un paràgraf llarg i fes que totes les línies estiguin justificades però l’última centrada.
  - Crea un altre paràgraf amb text-align-last: right.
- 1.3 direction + unicode-bidi
  - Escriu una frase i mostra-la amb lectura de dreta a esquerra.
  - Alterna dos paràgrafs: un amb rtl i l’altre amb ltr i compara visualment els resultats.
- 1.4 vertical-align
  - Crea tres <span> amb diferents mides i aplica vertical-align per aconseguir:
    - top
    - middle
    - bottom
      
# 2. Text decoration
- 2.1. text-decoration-line
  - Crea quatre paràgrafs amb les decoracions:
    - overline
    - underline
    - line-through
    - cap decoració (none)
- 2.2. text-decoration-color
  - Aplica un underline de color blau a un <h2>.
  - Aplica un line-through de color vermell a un paràgraf.
- 2.3. text-decoration-style
  - Crea una llista de 5 ítems i aplica a cadascun:
    - solid
    - double
    - dotted
    - dashed
    - wavy

- 2.4. text-decoration-thickness
  - Crea un text subratllat amb un gruix de 5px.
    
# 3. Transformació del text (text-transform)
- Fes un títol en MAJÚSCULES amb text-transform.
- Fes un paràgraf on totes les paraules comencin en majúscula (capitalize).
- Converteix un text que està escrit en majúscules a minúscules (lowercase).

# 4. Espaiat del text
- 4.1. text-indent
  - Crea un paràgraf i aplica-li una sagnia inicial de 40px.
- 4.2. letter-spacing
  - Fes un títol amb lletra molt separada (5px).
  - Apropa les lletres d’un paràgraf (-1px).
- 4.3. line-height
  - Crea un text amb:
    - line-height: 0.8
    - line-height: 1.3
    - line-height: 2
    - Compara quin és el més llegible.

# 5. Ombres de text (text-shadow)

- Fes un <h1> amb una ombra negra subtil (2px 2px).
- Fes un <h2> amb ombra vermella i difuminada (blur de 5px).
- Fes un text amb dues ombres:
  - una llima
  - una vermella

- Crea una frase que simuli un neó utilitzant diverses ombres.

# 6. Activitats de síntesi (barreja de propietats)
- Crea una targeta informativa (div) amb:
 - un títol centrat, en majúscules i amb ombra
 - un paràgraf justificat amb text-indent
 - un enllaç amb underline wavy de color blau
- Reprodueix un “cartell” amb:
 - Títol gegant amb text-shadow
 - Subtítol amb letter-spacing
 - Text alineat al centre
 - Decoració underline dashed
