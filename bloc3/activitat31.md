# Activitat 3.1

Entrega captures amb finestres creades amb Pygame y el codi utilitzat per tal que tinguin les següents característiques:

Cada finestra té la mateixa puntuació.

## Finestra 1:

Crea una finestra de 1000x1000 píxels del color que vulguis.

## Finestra 2:

Crea una finestra de 800x800 píxels amb una imatge que sigui una portada d'una peli que t'agradi i que ocupi la finestra sencera.

## Finestra 3:

Crea una finestra de 600 x 600 d'un color tropical amb una imatge d'un pokemon abaix a la dreta.

## Finestra 4:

Crea una finestra de 1200 x 600 d'un color fosc amb una imatge d'un personatge de videojoc centrat.
