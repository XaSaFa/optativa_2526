# Activitat joc meteorits utilitzant classes

Per aquesta activitat agafareu el fitxer de python amb el joc de meteors feta utilitzant classes, [que tens aqui](https://github.com/XaSaFa/optativa_2526/blob/main/bloc3/joc_meteorits_amb_classes.py).

I faràs modificacions per tal de que el joc tingui:

1. Dispars - La nau dispara cap amunt als meteors.
2. Turbo - Un botó de turbo que durant uns segons fa que la nau tingui més velocitat.
3. Escut - Un power up d'escut que fa invulnerable la nau durant uns segons.

Cada modificació val un 30%, els comentaris de codi explicant les vostres modificacions valen un 10%.
