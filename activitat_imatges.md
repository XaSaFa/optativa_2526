# Activitat imatges

- Crea una pàgina web nova amb 5 elements div.
- Cada element div ha de tenir unes característiques CSS diferents incloses dins d'una classe creada expressament.
  - El primer div té un color de fons escollit per tu amb format #RRGGBB.
  - El segon div té una imatge de fons amb estil de textura que recobreix tot l'element.
  - El tercer div té una imatge de fons repetida horitzontalment.
  - El quart div té una imatge de fons repetida verticalment (fes l'element prou alt perquè es vegi la repetició).
  - El cinquè div té una imatge de fons que a dalt al centre.

Fes-ho amb CSS Extern i entrega-ho com pagina1.html (i el css necessari).
Fes-ho amb CSS Intern i entrega-ho com pagina2.html.
Fes-ho amb CSS Inline (sense classes) i entrega-ho com pagina3.html.
